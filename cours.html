Composants d'un Microcontrôleur et leurs Rôles

COMPOSANT RÔLE PRINCIPAL RÔLES SPÉCIFIQUES EXEMPLE PRATIQUE
Flash Program Memory Stocker le programme exécutable
  • Contient le code machine
  • Stocke les constantes
  • Garde les adresses des fonctions
  • Contient le bootloader
 void setup() { pinMode(13,OUTPUT); }
Program Counter Pointer vers la prochaine instruction
  • Suivre l'exécution
  • Gérer les sauts
  • Séquencement du programme
 PC → adresse 0x100 (prochaine instruction)
Instruction Register Stocker l'instruction en cours
  • Garder l'instruction courante
  • Fournir au décodeur
 IR ← "ADD R1,R2" (instruction binaire)
Instruction Decoder Traduire les instructions
  • Décoder le code binaire
  • Activer les bonnes lignes de contrôle
 "1110 1101" → activation addition ALU
Control Lines Contrôler les composants
  • Activer/désactiver les unités
  • Synchroniser les opérations
 "Active ALU, active registre R1"
32x8 General Purpose Registers Calculs immédiats
  • Stockage temporaire ultra-rapide
  • Opérations arithmétiques
  • Adressage indirect
 R1 = R2 + R3 (1 cycle)
ALU Effectuer les calculs
  • Opérations arithmétiques
  • Opérations logiques
  • Comparaisons
 5 + 3 = 8 (calcul effectif)
Status and Control Surveiller l'état du CPU
  • Drapeaux de résultat
  • Contrôle des interruptions
  • État du processeur
 Z=1 (résultat zero), C=0 (pas de retenue)
Data SRAM Stocker les données variables
  • Variables du programme
  • Pile d'exécution
  • Données temporaires
 int compteur = 0; (en SRAM)
EEPROM Stockage persistant données
  • Sauvegarde de paramètres
  • Données de configuration
  • Historique utilisateur
 EEPROM.write(0, valeur);
I/O Lines Communication externe
  • Entrées numériques/analogiques
  • Sorties PWM
  • Communication
 digitalRead(A0), analogWrite(3, 128)
Interrupt Unit Gérer les interruptions
  • Priorité des interruptions
  • Vecteurs d'interruption
  • Gestion des urgences
 attachInterrupt(digitalPinToInterrupt(2), ISR, CHANGE);
SPI Unit Communication série haute vitesse
  • Communication avec périphériques
  • Transfert de données rapide
  • Protocole maître/esclave
 SPI.transfer(0x55);
Watchdog Timer Surveillance du système
  • Détection de plantage
  • Reset automatique
  • Sécurité système
 wdt_reset(); (reset du watchdog)
Analog Comparator Comparaison de tensions
  • Comparaison analogique
  • Détection de seuil
  • Conversion A/N rapide
 Comparaison tension A0 > A1
